How to play

You can move around the map in all directions, including diagonaly. This is done with the WASD keys. You can attack by holding the Left mouse button. Alternatevely, you can press the R key to make the character attack constantly. This will hit all the enemies in the shown range. After defeating enemies you get powerups. There are five different ones you can get, that are described below.
